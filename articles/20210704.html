在騰訊雲（DNSPod）如何對域名進行批量操作，你知道嘛？

發表於2021-07-04|更新於2021-07-04|教程
|字數總計:3.4k|閱讀時長:9分鐘|閱讀量:

你知道在騰訊雲（DNSPod）如何批量進行域名的批量註冊、批量轉入、批量續費、批量DNS修改、批量修改域名訊息、批量開啟或關閉禁止轉移鎖、批量開啟或關閉禁止更新鎖嘛？
本文較長，請在右側目錄單擊內容標題直接跳轉
PS：如您在操作中存在任何問題可以通過提交工單解決

下列操作均需在騰訊雲域名註冊控制台批量操作頁面（傳送門）進行操作。

批量域名註冊

通常來講若無必要的話不建議在此處批量註冊域名，比較騰訊雲域名註冊活動那麼多，在這裏註冊並不划算。

首先在批量操作頁面點擊域名註冊（進入批量操作頁面一般為域名註冊界面，通常無需點擊）

輸入域名


騰訊雲提供兩種輸入方式：輸入域名和上傳文件
輸入域名：通過在文本框中手動輸入或黏貼的方式進行域名的輸入，單次操作最多輸入200個
上傳文件：通過上傳文件的方式進行域名的輸入，單次上傳最多支持4000個域名

注意事項：
僅支持 txt/xls/xlsx 格式的文件。
上傳的文件大小不得超過2M，超過文件大小將無法進行上傳。
可以通過單擊【下載模板】，查看文件模板填寫格式。

選擇統一註冊年限

騰訊雲提供1-10年的域名註冊年限，可按需選擇

選擇域名訊息模版

可以選擇已有的模板也可以選擇新建訊息模板

選擇支付方式

騰訊雲提供以下三種支付方式，分別為：
使用賬戶餘額：提交後直接從您的賬戶餘額扣除費用，快速完成註冊，無需二次確認，不支持使用代金券
手動在線支付：提交後將自動創建訂單，您需要在訂單列表中逐一手動完成支付，支持使用代金券
域名特惠包：擇使⽤域名特惠包，僅滿⾜消耗條件的域名才可提交註冊，不⽀持使⽤代⾦券。提交後⽆需再次⽀付，將快速完成註冊（目前域名特惠包處於無貨狀態）

點擊下一步確認相關訊息並完成註冊

注意事項

域名為註冊型商品，域名註冊一旦提交並最終註冊成功，不支持退貨、退款，請您確認清楚後提交。
域名新注價格，請以域名價格頁當前顯示價格為準，請確認價格後再進行註冊。
如選擇使用賬戶餘額，將在提交註冊後依次嘗試註冊，一旦註冊成功則直接扣除賬戶餘額，請確保賬戶餘額充足。
如選擇手動在線支付，請提交後在訂單列表查看並及時完成支付，最終註冊成功之前將有可能被他人搶註。
如提交的域名列表中，部分域名無法完成註冊，剩餘域名將繼續嘗試註冊。
本功能僅提供普通域名註冊，暫不支持溢價詞、保留詞等特殊域名註冊。
域名註冊任務提交後，騰訊雲將依次向註冊局提交註冊申請。提交成功不代表域名註冊成功，騰訊雲不承諾域名最終註冊成功，是否註冊成功請以最終註冊結果為準。

批量域名轉入

首先在批量操作頁面點擊域名轉入

輸入域名


騰訊雲提供兩種輸入方式：輸入域名和上傳文件
輸入域名：通過在文本框中手動輸入或黏貼的方式進行域名和轉移密碼的輸入，單次操作最多輸入200個
上傳文件：通過上傳文件的方式進行域名和轉移密碼的輸入，單次上傳最多支持4000個域名

選擇域名訊息模版

可以選擇已有的模板也可以選擇新建訊息模板

注意：
必須已通過實名認證的訊息模版才可使用,騰訊雲新建模板審核一般情況下需要1個工作日內完成審核，部分可能需要等待3 - 7個工作日。

狀態設置

在 “狀態設置” 欄，請根據您的實際需求勾選以下選項：

注意：
根據國際互聯網域名與地址管理機構（ICANN）的 轉讓政策規定，修改域名訊息後60天內，禁止將域名從當前接入商轉出。
如您需將域名轉出騰訊雲，請您在轉入域名時，勾選【關閉 “60天內禁止轉移註冊商鎖定”】，否則轉入域名後60天內不能轉出。
PS:一般來講應該不會剛轉入就想轉出吧~

選擇支付方式並完成操作

騰訊雲提供以下兩種支付方式，分別為：
使用賬戶餘額：提交後直接從您的賬戶餘額扣除費用，快速完成註冊，無需二次確認，不支持使用代金券
手動在線支付：提交後將自動創建訂單，您需要在訂單列表中逐一手動完成支付，支持使用代金券
選擇支付方式並勾選 “我已閱讀並同意《騰訊雲域名註冊協議》和《騰訊雲私隱聲明》”後單擊下一步，確認相關訊息並完成操作

查看操作日誌

在【批量操作】管理頁面，選擇【操作日誌】頁簽。
選擇【操作任務】，並單擊【查看詳情】，即可查看您的域名轉入操作結果。。

注意事項：

域名轉入時必須選擇已實名認證的域名訊息模板。
域名轉入一般需要 5~7 個工作日，具體取決於註冊局審核時間。請提交後耐心等待，並留意域名列表。
如選擇使用賬戶餘額，將在提交後依次執行域名轉入，一旦轉入成功則直接扣除賬戶餘額，請確保賬戶餘額充足。
如選擇手動在線支付，請提交後在訂單列表查看並及時完成支付，避免影響轉入。
如域名有效期已達 10 年（最長期限），則轉入時仍需扣除費用但不增加年限。域名續費不足 45 天時不建議轉入，否則可能導致年限減少。
提交成功不代表域名轉入成功，是否轉入成功請以最終執行結果為準。域名一旦轉入成功，不支持撤銷、退款。
當前域名轉入僅支持非溢價詞轉入，為您帶來的不便敬請諒解。

批量域名續費

首先在批量操作頁面點擊域名續費

輸入域名


騰訊雲提供兩種輸入方式：輸入域名和上傳文件
輸入域名：通過在文本框中手動輸入或黏貼的方式進行域名的輸入，單次操作最多輸入200個
上傳文件：通過上傳文件的方式進行域名的輸入，單次上傳最多支持4000個域名

選擇統一註冊年限

騰訊雲提供1-9年的域名續費年限，可按需選擇，域名有效期最長不超過10年

選擇支付方式並完成操作

騰訊雲提供以下兩種支付方式，分別為：
使用賬戶餘額：提交後直接從您的賬戶餘額扣除費用，快速完成註冊，無需二次確認，不支持使用代金券
手動在線支付：提交後將自動創建訂單，您需要在訂單列表中逐一手動完成支付，支持使用代金券

注意事項：

如選擇使用賬戶餘額，將在提交後依次執行域名續費，一旦續費成功則直接扣除賬戶餘額，請確保賬戶餘額充足。
如選擇手動在線支付，請提交後在訂單列表查看並及時完成支付。
本頁面僅根據普通域名續費價格預估，白金域名存在價格波動情況，請以最終結算價格為準，如介意請勿提交。
提交成功不代表域名續費成功，是否續費成功請以最終執行結果為準。域名一旦續費成功，不支持撤銷、退款。
域名最長有效期不可超過 10 年。
域名續費實際價格請以最終訂單結算價格為準。

批量DNS修改

首先在批量操作頁面點擊DNS修改

輸入域名


騰訊雲提供兩種輸入方式：輸入域名和上傳文件
輸入域名：通過在文本框中手動輸入或黏貼的方式輸入需要修改DNS的域名，單次操作最多輸入200個
上傳文件：通過上傳文件的方式進行域名的輸入，單次上傳最多支持4000個域名

選擇或輸入DNS伺服器

修改 DNS 伺服器可以選擇兩種方式進行操作，分別是使用 DNSPod （推薦）和自定義 DNS：
選擇使用 DNSPod （推薦）：自動為域名匹配當前套餐的 DNSPod 伺服器的 DNS 地址。

選擇使用 DNSPod 會匹配當前解析所屬套餐的 DNS 地址。

選擇自定義 DNS：填寫您需要為域名設置的 DNS 伺服器地址。

需要在騰訊雲進行解析的域名，修改 DNS 伺服器地址請參考 修改 DNS 伺服器。
自定義的 DNS 伺服器域名不能是私建的 DNS 伺服器域名，必須是解析商的權威 DNS 伺服器域名。

查看操作日誌

在【批量操作】管理頁面，選擇【操作日誌】頁簽。
選擇【操作任務】>【DNS 修改】，並單擊【查看詳情】，即可查看您的域名 DNS 修改結果。

注意事項：

修改 DNS 伺服器，表示更換其他 DNS 伺服器負責當前域名的解析服務。修改錯誤可能影響當前域名正常解析，請謹慎操作。
如選擇使用 DNSPod 伺服器，請確保您已在 DNSPod 添加相關域名解析，否則可能導致域名無法正常解析。
如選擇使用 DNSPod 伺服器，系統將分別為各個域名匹配當前最佳伺服器地址，如無法識別則使用免費版。是否符合預期，請自行確認。
任務提交後系統將開始執行，因域名狀態各異，可能存在部分域名修改失敗。是否修改成功，請以執行結果為準。
DNS 伺服器地址修改成功後，因各地運營商 DNS 緩存機制不同，最長需要 72 小時的全球生效時間，請您耐心等待。

批量域名訊息修改

首先在批量操作頁面點擊域名訊息修改

輸入域名


騰訊雲提供兩種輸入方式：輸入域名和上傳文件
輸入域名：通過在文本框中手動輸入或黏貼的方式進行域名的輸入，單次操作最多輸入200個
上傳文件：通過上傳文件的方式進行域名的輸入，單次上傳最多支持4000個域名

選擇域名訊息模版

可以選擇已有的模板也可以選擇新建訊息模板

注意：
必須已通過實名認證的訊息模版才可使用,騰訊雲新建模板審核一般情況下需要1個工作日內完成審核，部分可能需要等待3 - 7個工作日。

勾選同意協議並完成操作

在 “狀態設置” 欄，您可以實際情況勾選【關閉 “60天內禁止轉移註冊商鎖定”】，並勾選 “我已閱讀並同意《騰訊雲域名註冊協議》和《騰訊雲私隱聲明》”後單擊下一步,確認相關訊息並完成操作

注意事項

域名訊息修改必須選擇已實名認證的域名訊息模板。
域名訊息修改審核時間一般需要 3~7 個工作日，具體取決於相關注冊局審核時間。請提交後耐心等待，並留意域名狀態。
根據 ICANN 相關政策要求，域名所有者的電子郵箱地址必須確保真實有效。請您確保所選擇的域名訊息模板資料填寫正確。
域名訊息修改審核期間不影響該域名的正常使用及 DNS 解析，域名訊息審核結果將通過郵件通知，請您留意。
任務提交成功後，系統將依次執行操作，操作完成後需要等待註冊局審核，是否修改成功請以審核結果為準。
