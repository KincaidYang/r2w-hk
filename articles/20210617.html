DNSPod出品的DoH客戶端，一鍵為電腦打開DoH服務

發表於2021-06-17|更新於2021-10-17|教程
|字數總計:346|閱讀時長:1分鐘|閱讀量:

DoT 全稱是 DNS over TLS，它使用 TLS 協議來傳輸 DNS 協議。 TLS 協議是目前互聯網最常用的安全加密協議之一，我們訪問 HTTPS 的安全基礎就是基於 TLS 協議的。相比於之前使用無連接無加密的 UDP 模式， TLS 本身已經實現了保密性與完整性。而DoH 全稱是 DNS over HTTPS，它使用 HTTPS 來傳輸 DNS 協議。 DoH 的安全原理與 DoT 一樣，他們之間的區別只在於：DoH 有了 HTTP 格式封裝，更加通用。

DNSPod DoT&Doh服務：官網傳送門
DoT地址：dot.pub
DoH地址：https://doh.pub/dns-query
專業版：傳送門

DNSPod DoH客戶端的Windows版現已在騰訊電腦管家中上架了，大家可以直接去軟件管家→“網絡”分類下，按照發佈時間排序查找，同時頁面下方提供單文件鏈接，可直接下載。後續這個工具還將持續更新，把公共DNS的最新能力展現出來，歡迎大家體驗！ （部分殺毒軟件可能會提示風險，請忽略！）
